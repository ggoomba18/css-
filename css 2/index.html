Clases en párrafos

El selerane de clase aplica sus estilos a todos los elementos HTML. que incluyan dicho nombre de clase. Para declarar este selector CSS incluimos el nombre de clase precedido de un punto (.)

A continuación crestemos una clase para modificar el color, fuente, tamaño y alineación de muestro párrafos

Ejemplo CSS:

.justificado{
        font-family: Arial, Helvetica, sans-serif
        color: #e3e3
        text-align: justify;
        }

En el código HTML aplicamos la clase de la siguiente forma

        < p class"justificado"> < /p >

clases en listas

Podemos cambiar los estilos de las viñetas en nuestras listas no ordenadas creando una clase y aplicando el estilo list-style

Listas no ordenadas

Para las listas no ordenadas podemos elegir entre los siguientes estilos

            .ul nada {
        
            list-style : none;
            
            }
            .ul_circle{
        
            list-style: circle;
            
            }
            .ul_square{
        
            list - s * t : square;
            
            }
            .ul_imagen {
        
            list-style-image:url("img/ipn.GIF");
            
            }
